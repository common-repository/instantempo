== Installation ==

Download, Upgrading, Installation:

Upgrade

1. First deactivate instantempo
1. Remove the `instantempo` directory

**Install**

1. Unzip the `instantempo.zip` file. 
1. Upload the the `instantempo` folder (not just the files in it!) to your `wp-contents/plugins` folder. If you're using FTP, use 'binary' mode.

**Activate**

1. In your WordPress administration, go to the Plugins page
1. Activate the instantempo plugin and then go to your widget page.

If you find any bugs or have any ideas, please mail me.
